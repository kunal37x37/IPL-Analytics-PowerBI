# IPL Analytics Dashboard (Power BI)

## Project Overview

The **IPL Analytics Dashboard** is a data visualization project developed using **Microsoft Power BI** to analyze Indian Premier League (IPL) cricket data from **2008 to 2025**. The main objective of this project is to transform large volumes of IPL match statistics into interactive visual insights that help users understand team performance, player statistics, and match trends.

This dashboard provides a comprehensive analysis of IPL data including total runs, wickets, matches played, top performing batsmen, leading bowlers, and team performance across seasons. Using interactive filters and visualizations, users can explore cricket statistics dynamically and gain meaningful insights from historical IPL data.

---

## Features

* Interactive **Power BI Dashboard**
* Analysis of **Total Matches, Runs, Wickets, Fours, and Sixes**
* **Top Batsman Analysis** based on total runs
* **Top Bowler Analysis** based on total wickets
* **Team Performance Analysis**
* **Orange Cap List (Highest Run Scorer)**
* **Purple Cap List (Highest Wicket Taker)**
* Match summary and points table visualization
* Filters for **Year, Team, Venue, Batsman, and Bowler**
